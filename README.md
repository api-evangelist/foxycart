# FoxyCartThis is a repo for managing machine readable artifacts for the [FoxyCart APIs](https://www.foxy.io/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [FoxyCart APIs](https://www.foxy.io/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.foxy.io/)- [Developer](https://www.foxy.io/)- [Twitter](https://twitter.com/foxy_io)- [GitHub](https://github.com/FoxyCart)The [APIs.json](https://github.com/api-evangelist/foxycart/blob/master/apis.json) index provides links to all the relevant links for the FoxyCart APIs, including the OpenAPI and Postman if they are present.
